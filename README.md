# GPS Demo Leaflet

### 사용자의 위치 정보를 leaflet 웹 지도로 표시

### [gps App](https://github.com/WooJin1030/HAN-GPS_DemoApp)

## 기술

- React
- Leaflet.JS

## 기능

- 사용자별 위치를 마커로 표시
- 각 마커들의 팝업에서 날짜, 시간 정보 표시
- 사용자별 위치 마커들은 시간 순에 따라 직선의 경로로 표시
- 원 또는 다각형의 범위 설정
- 범위 밖으로 나갈 때 마커가 경고 표시로 바뀜
- 범위 밖으로 나갈 때 팝업창에 경고 표시 뜸
